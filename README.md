# HRM System (Flask)

A simple Human Resource Management System built using Flask and SQLite.

## Features

- Add Department
- View Departments
- Search Departments
- Edit Department
- Soft Delete Department
- Dashboard UI

## Tech Stack

- Python (Flask)
- SQLite
- HTML / Bootstrap
- Git & GitHub

## Project Structure

HRM_project
│
├── app.py
├── requirements.txt
│
├── templates
│   ├── index.html
│   ├── add_department.html
│   └── edit_department.html
│
└── static
    ├── style.css
    ├── script.js

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the project:

python app.py

Open in browser:

http://127.0.0.1:5000
